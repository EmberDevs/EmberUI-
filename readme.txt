Installation
Download EmberUI:

Get the latest version of the EmberUI executor from Github or File.Io
Extract Files:

Unzip the downloaded file to a folder on your computer.
Run the Executor:

Open the EmberUIClient.exe file to launch the application.
Getting Started
Open Roblox:

Start the Roblox game you want to use with EmberUI.
Paste Your Script:

Copy a script from a trusted source and paste it into the input box in EmberUI.
Execute the Script:

Click the "Execute" button to run the script in the game.
Customization
Settings:
Adjust settings such as themes and font sizes in the settings menu for a better experience.
